# シンプルブログサイト

シンプルなテキスト主体のブログサイトです。マークダウンファイルからコンテンツを読み込む静的サイトです。

## 特徴

- ミニマルなデザイン
- テキスト主体のシンプルなレイアウト
- マークダウンファイルでコンテンツを管理
- レスポンシブ対応
- 軽量で高速

## ファイル構成

- `index.html` - メインのHTMLファイル
- `style.css` - スタイルシート
- `script.js` - JavaScriptファイル（マークダウンファイルを読み込む）
- `posts.md` - 投稿データ（マークダウンファイル）
- `music.md` - 音楽作品データ（マークダウンファイル）

## 使い方

### Netlifyにデプロイ

1. GitHubリポジトリにプッシュ
2. Netlifyでリポジトリを接続
3. ビルドコマンドは不要（静的サイト）
4. 公開ディレクトリはルート（`/`）

### ローカルで開く

1. ローカルサーバーを起動（マークダウンファイルを読み込むため必要）
   - Python 3の場合: `python -m http.server 8000` を実行し、`http://localhost:8000` にアクセス
   - Node.jsの場合: `npx http-server` を実行
2. ブラウザで `http://localhost:8000` にアクセス

**注意**: `index.html`を直接ブラウザで開くだけでは、マークダウンファイルが読み込めません。

### 投稿を追加・編集する

`posts.md`ファイルを編集してください。形式は以下の通りです：

```
2024/01/20 新しい投稿 [日記]
これは記事の本文です。
複数行の文章も書けます。

**太字**や*イタリック*も使えます。
[リンク](https://example.com)も追加できます。

## 見出し
見出しも使えます。

---
2024/01/18 別の投稿 [カテゴリ名]
別の記事の本文です。
記事の区切りには `---` を使います。
```

**形式の説明:**
- 日付形式: `YYYY/MM/DD`
- タイトルの後に `[カテゴリ名]` を追加するとカテゴリが表示されます（省略可能）
- タイトルの次の行から本文を書けます
- 記事の区切りには `---` を使います
- マークダウンの基本的な記法（太字、イタリック、リンク、見出し）が使えます

### 音楽作品を追加・編集する

`music.md`ファイルを編集してください。形式は以下の通りです：

```
2024/01/20 新しい楽曲
この楽曲についての説明を書けます。

制作の経緯や、使用した楽器、コンセプトなどを記録できます。
---
2024/01/18 別の楽曲
別の楽曲について。
```

**形式の説明:**
- 日付形式: `YYYY/MM/DD`
- タイトルの次の行から本文を書けます
- 記事の区切りには `---` を使います
- 本文は省略可能です（タイトルのみでもOK）

## カスタマイズ

### スタイルを変更する

`style.css`を編集してデザインをカスタマイズできます。

### タイトルを変更する

`index.html`の`<title>`タグと`<h1>`タグを編集してください。

### 記事の詳細を表示する

記事一覧のタイトルをクリックすると、モーダルで記事の詳細（本文）が表示されます。
モーダルを閉じるには、×ボタンをクリックするか、モーダルの外側をクリックしてください。

### セクションを追加する

1. `index.html`に新しいセクションを追加
2. ナビゲーションにリンクを追加
3. 必要に応じて新しいマークダウンファイルを作成し、`script.js`で読み込む

## トラブルシューティング

### Netlifyで投稿一覧が表示されない

- `posts.md`と`music.md`がリポジトリに含まれているか確認してください
- ブラウザのコンソールでエラーメッセージを確認してください
- マークダウンファイルの形式が正しいか確認してください（日付形式: `YYYY/MM/DD`）
